Perceptual Visual Interactive Learning

Supervised learning methods are widely used in machine learning. However, the lack of labels in existing data limits the application of these technologies. Visual interactive learning (VIL) compared with computers can avoid semantic gap, and solve the labeling problem of small label quantity (SLQ) samples in a groundbreaking way.

Tutorial:

1. 图中的点分为了若干类， 实验的流程便是通过圈选将 已有标记（有颜色）和 数据分布 来标记 待标记的点（无色）

2. 操作分为两种方式，SIDR(Single Interactive Dimensionality Reduction) 和 MIDR(Multiple Interactive Dimensionality Reduction)

3. SIDR便于介绍，选择图中相近的数据流行，之后确认选择。所选区域中所有未标记点被标记为 [区域中数量最多的已标记点类别]

4. MIDR相对复杂，当用户难以确定某一类无标记点C的归属时（假设有近邻标记点A 和 近邻标记点B），将ABC一起圈选出来，此时系统判定对这些数据进行二次处理。 用户会看到二次处理后的分布，在这个分布中判断数据应该属于哪个类别
